# Modules

User Interface Module  
Handles text input and language selection.  
Displays translated output to the user.  

Translation Engine Module  
Processes translation requests.  
Communicates with the AI model.  

API Integration Module  
Manages API authentication securely.  
Handles communication with Google Gemini API.  

Error Handling Module  
Manages invalid inputs.  
Handles API and system errors gracefully.
