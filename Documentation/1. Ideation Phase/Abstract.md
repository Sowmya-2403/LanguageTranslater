# Abstract

The AI-Powered Multi-Language Translator is a web-based application.  
It is designed to translate text from one language to another.  
The system uses Google Gemini generative AI for translation.  
The application provides fast and accurate translation results.  
It supports multiple international languages.  
The user interacts with the system through a simple Streamlit interface.  
The project demonstrates the real-world use of artificial intelligence.
